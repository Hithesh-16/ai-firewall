Visual Studio Extension Skeleton

This folder will contain a VSIX project (C#) implementing a Roslyn CompletionProvider and a ToolWindow using WebView2.

Steps to implement:
- Create new VSIX project in Visual Studio (use .NET Framework or .NET 6+ depending on SDK)
- Implement CompletionProvider using Roslyn APIs
- Implement ToolWindow hosting WebView2 for chat UI
- Package as .vsix for publishing to Visual Studio Marketplace
